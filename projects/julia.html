Julia for Scientific Computing

What is Julia?

Julia is a general purpose dynamic programming language that supports mathematical and statistical operations similar to R and vectorized numerical functions similar to MATLAB all with python-like syntax - making ideal for high-performance scientific computing. Julia’s current ecosystem is mature enough to build and deploy analytics web applications, making it a serious contender to R and Python.

Julia has some unique features unavailable in current langauges used for scientific computing, including native support for parallel and distributed computing and an extensive library - most of the popular computing libraries available in python such as SciKit Learn and Tensorflow are available in Julia’s package manager and python-only packages can be utilized via PyCall.

Performance: Julia vs. Python vs. Scala

To get a better idea of the performance gains from Julia, lets compare it to other popular langauges used in Machine Learning / Data Science. Below is a monte carlo simulation to estimate the value of pi implemented in Scala, Python, and Julia. The results show that Julia is by far the most efficient.

Scala Elapsed Time: 140.041331813 sec

import scala.util.Random
def mcCount(nsamples: Int): Int = {
    val randomX = new Random
    val randomY = new Random
    var acc = 0
    for (i <- 1 until nsamples) {
        val x = randomX.nextDouble 
        val y = randomY.nextDouble 
        if (x*x + y*y < 1) acc= acc + 1
    }
    acc
}
def monte_carlo_pi(iter: Int): Double = 4.0 * mcCount(nsamples) / nsamples
val nsamples = scala.math.pow(10,10)
val t0 = System.nanoTime()
val result = monte_carlo_pi(nsamples.toInt)
val t1 = System.nanoTime()
val elapsed_time = (t1-t0)/scala.math.pow(10,9)
println("Elapsed Time: " + elapsed_time +"sec")

Python (w/Numba) Elapsed Time: 175.5762140750885 sec

import time
from numba import njit
import random
@njit(nopython=True, parallel=True)
def monte_carlo_pi(nsamples):
    acc = 0
    for i in range(nsamples):
        x = random.random()
        y = random.random()
        if (x ** 2 + y ** 2) < 1.0:
            acc += 1
    return print(4.0 * acc / nsamples)
t0 = time.time()
monte_carlo_pi(10**10)
t1 = time.time()
print("Elapsed Time: ", (t1- t0),"sec")  

result = 3.1415756664

Julia Elapsed Time: 52.1004299293578 sec
